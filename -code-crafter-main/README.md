## 🌟 Features

### 🏥 Medical Services
- 📅 Online/Offline Appointment Scheduling
- 📝 Digital Prescription Management
- 🚚 Home Delivery Services
- 🔍 Smart Doctor Search

### 👥 Community Engagement
- 👨‍⚕️ Doctor Communities
- 👥 User Forums
- 📰 Health Blogs
- 🩸 Blood Donation System

### 🔧 Technical Features
- 📍 Location-based Disease Alerts
- 🔐 Secure Authentication (JWT)
- 💬 Real-time Communication
- 📱 Responsive Design

## 🛠️ Tech Stack

### Backend
- Spring Boot Microservices
- Redis Caching
- PostgreSQL Database
- Kafka for Real-time Communication

### Frontend
- React.js
- Redux/Context API
- Material-UI
- Responsive Design

### DevOps
- Kubernetes
- Jenkins
- Docker
- CI/CD Pipeline

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/Learnathon-By-Geeky-Solutions/-code-crafter.git
```

2. Install dependencies
```bash
# Frontend
cd frontend
npm install

# Backend
cd ../xenonvita
mvn install
```

3. Start development
```bash
# Frontend
npm start

# Backend
mvn spring-boot:run
```
